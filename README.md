# MALSyncArabic

**MALSyncArabic** is a specialized fork of MAL-Sync focused on Arabic anime and manga sites. It integrates MyAnimeList/AniList/Kitsu/Simkl with Arabic anime and manga websites, providing automatic episode/chapter tracking.

This extension enables you to use your MyAnimeList/AniList/Kitsu/Simkl anime and manga lists as a centralized bookmarks system for supported Arabic sites.

## What more can it do?

MyAnimeList

- Direct links to the supported websites
- Links to the next episode
- Enlarge thumbnails in many places
- Adds missing watching status tags like in the related anime section
- Estimated time to next episode and estimated current episode number (JP release time)

miniMAL (Extension Popup)

- Allows to browse, search and edit MyAnimeList anime/manga everywhere
- Access MAL Bookmarks everywhere
- MAL-Sync's Settings
